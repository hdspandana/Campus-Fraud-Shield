# AI Disclosure

## Project Name
Campus Fraud Shield 🛡️

## Purpose of AI Usage
AI tools and machine learning models were used to assist in the development of this project for scam detection and explainability.

---

## AI Tools Used

### 1. OpenAI ChatGPT
Used for:
- Debugging Python and Streamlit errors
- Improving UI/UX design
- Generating sample datasets and testing messages
- Assisting with code structure and documentation
- Helping integrate FAISS, semantic similarity, and ML pipelines

### 2. Sentence Transformers
Model used:
- `all-MiniLM-L6-v2`

Purpose:
- Semantic scam message classification
- Similarity matching between scam messages

### 3. FAISS (Facebook AI Similarity Search)
Used for:
- Fast similarity search on historical scam reports
- Community-based scam pattern matching

---

## Human Contribution

The team members:
- Designed the project architecture
- Integrated all modules manually
- Developed Streamlit frontend and backend integration
- Created fraud detection logic
- Tested and validated outputs
- Prepared documentation, PPT, and deployment

AI-generated outputs were reviewed, modified, and validated before inclusion in the final project.
